# flet-desktop-drop
FletDesktopDrop control for Flet

## Installation

Add dependency to `pyproject.toml` of your Flet app:

* **Git dependency**

Link to git repository:

```
dependencies = [
  "flet-desktop-drop @ git+https://github.com/MyGithubAccount/flet-desktop-drop",
  "flet>=0.27.6",
]
```

* **PyPi dependency**  

If the package is published on pypi.org:

```
dependencies = [
  "flet-desktop-drop",
  "flet>=0.27.6",
]
```

Build your app:
```
flet build macos -v
```
